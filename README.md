# Model training on Student Depression Dataset

Mental health among university and college students has become a critical global concern. Depression is often driven by academic pressure, financial stress, and poor lifestyle habits. This project aims to make a deep analysis of the factors and parameters that cause student depression. By implementing a machine learning pipeline, we seek to automatically predict if a student is suffering from depression, acting as an early-warning system to proactively flag high-risk individuals for counseling. Early and accurate identification of at-risk students allows universities and counselors to intervene proactively before conditions worsen.

What the project does:
* Analyses 27,000+ student records with features spanning academic pressure, sleep habits, dietary habits, financial stress, CGPA, family mental health history, and more.
* Applies comprehensive EDA to identify the most impactful risk factors.
* Trains and evaluates six supervised classification models, using Optuna for hyperparameter tuning and SMOTE for oversampling to handle class imbalance.
* Applies K-Means clustering to discover natural groupings in the data without labels.
* Compares all models using Accuracy, Precision, Recall, F1-Score, AUC-ROC, and Confusion Matrices.

Suicide rates and mental health crises among students have increased sharply in recent years. Traditional screening relies on manual surveys administered infrequently. An ML-based system can process existing student records and flag high-risk individuals continuously, enabling a data-driven early-warning system for student wellbeing programs.
